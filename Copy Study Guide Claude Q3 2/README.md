# HTML Learning Objects Workspace

Create interactive HTML-based educational content.

## Quick Start

1. **Open the quiz template:**
   ```bash
   open templates/quiz_template.html
   ```

2. **Request custom content:**
   Just tell Claude what you need:
   - "Create an interactive quiz on [topic]"
   - "Generate flashcards from my notes"
   - "Build an interactive lesson about [subject]"

## Features

- ✅ Interactive quizzes with instant feedback
- ✅ Flashcard applications
- ✅ Self-contained HTML files (work offline)
- ✅ Mobile responsive
- ✅ No installation required - just open in browser

## Directory Structure

- `lessons/` - Interactive lesson modules
- `quizzes/` - Quiz files
- `flashcards/` - Flashcard apps
- `visualizations/` - Interactive diagrams
- `templates/` - Base templates for customization
- `assets/` - Shared CSS/JS/images

## Examples

See `templates/quiz_template.html` for a complete interactive quiz example.

Claude can customize this or create entirely new learning objects based on your content.
