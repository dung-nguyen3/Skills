JavaScript Toggle Function:
```javascript
function toggleAccordion(element) {
    var accordion = element.parentElement;
    var wasActive = accordion.classList.contains('active');

    // Toggle the clicked accordion
    if (wasActive) {
        accordion.classList.remove('active');
    } else {
        accordion.classList.add('active');
    }
}
```

IMPORTANT - Condition-Specific Physical Exam Findings (Tabs 1-3):
For each condition's Physical Exam Findings section, specify the EXACT exam name for each finding. Do not just list findings alone - list the exam name in bold followed by the finding with specific details.

Format: ☐ **Exam Name:** Finding (specific details/which tests)

Examples:
• Instead of: ☐ Hyporeflexia or areflexia
  Write: ☐ **Deep Tendon Reflexes:** Hyporeflexia or areflexia (test patellar L4, achilles S1)

• Instead of: ☐ Dermatomal sensory changes
  Write: ☐ **Sensory Testing:** Dermatomal sensory changes (test C5-T1 or L3-S1 dermatomes)

• Instead of: ☐ Myotomal weakness
  Write: ☐ **Motor Testing:** Myotomal weakness (test specific myotomes: hip flexion L3, knee extension L4, great toe extension L5, ankle plantarflexion S1)

• Instead of: ☐ Pain with palpation
  Write: ☐ **Palpation:** Point tenderness over spinous process

Common Exam Names to Use:
- **Inspection:** Posture, gait, deformity, skin changes, muscle spasm
- **Palpation:** Spinous processes, paraspinal muscles, facet joints, SI joints
- **Percussion:** Fist percussion over spine
- **ROM Testing:** Flexion, extension, lateral bending, rotation
- **Positional Testing:** Pain with sitting, standing, walking
- **Sensory Testing:** Dermatomes (specify: C5-T1 for upper extremity, L3-S1 for lower extremity, S2-S5 for saddle area)
- **Motor Testing:** Myotomes (specify which: shoulder abduction C5, elbow flexion C6, hip flexion L3, knee extension L4, great toe extension L5, ankle plantarflexion S1)
- **Deep Tendon Reflexes:** Biceps C5-6, Brachioradialis C6, Triceps C7, Patellar L4, Achilles S1
- **Babinski Reflex:** Pathologic reflex testing for UMN lesion
- **Clonus Test:** Rapid ankle dorsiflexion to test for UMN lesion
- **Lhermitte Sign:** Neck flexion causing electric shock sensation
- **Gait Assessment:** Walking pattern, heel/toe walking, tandem gait
- **Muscle Tone:** Assessment for spasticity/hypertonia/hypotonia
- **Specialized Tests:** Straight Leg Raise, Spurling Maneuver, One-Leg Hyperextension Test, Walking Test (for claudication), Valsalva Maneuver
- **Rectal Exam:** Anal sphincter tone (for cauda equina assessment)
- **Neurologic Exam:** When normal, state "Normal (sensory, motor, DTRs all intact)"
