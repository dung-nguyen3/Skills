# Quick Reference - Which Command to Use

## By Output Format

| Want This? | Use This Command |
|------------|------------------|
| Word study guide | `/LO-word` |
| Excel drug chart (4 tabs) | `/4-tab-excel` |
| Excel single-sheet overview | `/master-chart-excel` |
| Excel side-by-side comparisons | `/key-comparisons-excel` |
| Anki flashcards | `/anki` |
| HTML study guide | `/LO-html` |
| HTML drug reference | `/drugs-html` |
| Drug autobiography stories | `/autobiography-trick` |

## Clinical Assessment

| Want This? | Use This Command |
|------------|------------------|
| Clinical H&P (Excel) | `/clinical-assessment-excel` |
| Clinical H&P (HTML) | `/clinical-assessment-html [file] [chief complaint]` |

## Bundles

| Want This? | Use This Command |
|------------|------------------|
| Word + Excel + Anki | `/word-excel-anki` |
| All formats at once | `/create-all` |
| Interactive menu | `/study-guides` |

## Batch

- Multiple files: `"file1.txt;file2.txt"`
- Whole folder: `"/path/to/folder/"`
