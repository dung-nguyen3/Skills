# Claude Instructions: Interactive HTML Drug Reference Charts

Convert drug information/pharmacology notes into ONE self-contained HTML drug reference chart. Use ONLY uploaded content. Inline CSS/JS only.

---

## CRITICAL REQUIREMENTS

1. **SOURCE ONLY**: Use ONLY file content. Do NOT add external pharmacology knowledge. If missing: `[Information not provided in source material]`

2. **EXTRACT ALL DRUGS**: Search for drug names in learning objectives, medication lists, treatment protocols. Include generic and brand names.

3. **SINGLE HTML FILE**: All CSS in `<style>`, all JS in `<script>`. No external dependencies. Must work offline.

4. **THREE STYLING TYPES**:
   - **HIGHLIGHTS**: Gradient backgrounds wrapping text ‚Üí `<span class="highlight-purple">Mechanism text</span>`
     Types: `.highlight-green/purple/blue/red/yellow/teal`
   - **COLOR BOXES**: Full-width containers ‚Üí `<div class="high-yield-box">Content</div>`
     Types: `.high-yield-box`, `.clinical-pearl`, `.critical-box`, `.mnemonic-box`

5. **MASTER CHART COLUMNS IF applicable** (adapt to learning objectives and drug class):
 Infection type (example: Cancer, antiviral, HIV, etc.) | Drug Class | Drug Name | Route | Mechanism | | Adverse Effects| Contraindications (badge)| Resistance if applicable | Drug Interactions | Special Considerations (things to test for, Adjuncts to mitigate adverse side effects, etc.)|  Additional columns as needed


6. **CREATE MNEMONICS**: Memory aids for drug lists, side effects, mechanisms, contraindications, etc. 
**Research: Good memory tricks/mnemonics**
---

## üîç STEP 1: EXTRACT & ORGANIZE

**Find Drugs**: Search for "Describe pharmacology of...", drug lists, treatment protocols, medication tables. Extract: drug name, class, mechanism, indications, dosing, contraindications, side effects, monitoring, pearls.

**Create Mnemonics**: For drug class members, side effect lists, indications, contraindications, mechanism steps. Use acronyms, name patterns (e.g., "-olol" = beta blockers).

**Organize with**:
- **Drug Classes Tab**: Group drugs by class with overview, key similarities, key differences
- **Master Chart Tab**: All drugs in sortable table
- **Quick Reference Tab**: High-yield cards (first-line, warnings, monitoring, mnemonics)

**When comparing, "Make the items being compared as ROWS and the comparison categories as COLUMNS"

// Main tab switching
function showMainSection(sectionId) {
    document.querySelectorAll('.main-section').forEach(section => {
        section.classList.remove('active');
    });
    document.querySelectorAll('.main-tab').forEach(tab => {
        tab.classList.remove('active');
    });
    document.getElementById(sectionId + '-section').classList.add('active');
    event.target.classList.add('active');
}

// Sub-tab switching
function switchTab(tabId) {
    const parentSection = event.target.closest('.main-section');
    parentSection.querySelectorAll('.content-section').forEach(section => {
        section.classList.remove('active');
    });
    parentSection.querySelectorAll('.tab').forEach(tab => {
        tab.classList.remove('active');
    });
    document.getElementById(tabId).classList.add('active');
    event.target.classList.add('active');
}

// Table sorting
function sortTable(columnIndex, tableId) {
    const table = document.getElementById(tableId);
    const tbody = table.querySelector('tbody');
    const rows = Array.from(tbody.querySelectorAll('tr'));
    
    const sortedRows = rows.sort((a, b) => {
        const aText = a.cells[columnIndex].textContent.trim();
        const bText = b.cells[columnIndex].textContent.trim();
        return aText.localeCompare(bText);
    });
    
    tbody.innerHTML = '';
    sortedRows.forEach(row => tbody.appendChild(row));
}

// Initialize
document.addEventListener('DOMContentLoaded', function() {
    const firstMainTab = document.querySelector('.main-tab');
    const firstMainSection = document.querySelector('.main-section');
    if (firstMainTab && firstMainSection) {
        firstMainTab.classList.add('active');
        firstMainSection.classList.add('active');
    }
    
    document.querySelectorAll('.main-section').forEach(section => {
        const firstTab = section.querySelector('.tab');
        const firstContent = section.querySelector('.content-section');
        if (firstTab && firstContent) {
            firstTab.classList.add('active');
            firstContent.classList.add('active');
        }
    });
});

## üìä STEP 5: STYLING & COMPARISON GUIDELINES

### **Styling Types & Usage**

**HIGHLIGHTS**: High-yield, contraindications
**COLOR BOXES** (Drug Classes Tab): Class overview, similarities, differences, mnemonics

### **Color-Coded Box Content**

**HIGH-YIELD** (`.high-yield-box`, no highlights here): Most testable facts, common exam questions, must-memorize values, distinguishing features, key thresholds

**CLINICAL PEARL** (`.clinical-pearl`): Practical tips, common pitfalls, "If X think Y" associations, prescribing wisdom

**CRITICAL/EMERGENCY** (`.critical-box`): Emergency conditions, never-miss diagnoses, red flags, urgent consultation criteria, black box warnings, serious drug interactions

**MNEMONIC** (`.mnemonic-box`): Memory aids for drug lists, side effects, mechanisms, contraindications

### **Comparison Strategies**

Use **SAME COLOR** for SHARED features across drugs (e.g., all ACE-I contraindicated in pregnancy = red)
Use **DIFFERENT COLORS** for UNIQUE features (e.g., different mechanisms = different colors)
Use **GREEN** for preferred/better options
Use **RED** for warnings/worse features
Use **PURPLE** for mechanism groupings
Use **BLUE** for indication patterns

## ‚úÖ WORKFLOW & CRITICAL REMINDERS

**Process**: Analyze file ‚Üí Extract all drugs ‚Üí Group by class ‚Üí Build Tab 1 (classes with similarities/differences) ‚Üí Build Tab 2 (master chart with all columns) ‚Üí Build Tab 3 (quick reference) ‚Üí Apply badges/highlights/boxes ‚Üí Create mnemonics ‚Üí Deliver single HTML

**Critical Rules**:
1. SOURCE ONLY - No external pharmacology knowledge
2. THREE styling types: Badges (routes, drug choice), Highlights (mechanisms, effects), Boxes (comparisons, mnemonics)
3. Master Chart columns: Drug Class, Drug Name, Route, Bactericidal/static, Mechanism, Spectrum, Drug of Choice, Adverse Effects, Contraindications, Resistance
4. Drug Classes tab: Class overview + Key similarities + Key differences table + Mnemonics
5. Create mnemonics for drug patterns, side effects, contraindications
6. Use color coding consistently to show patterns (same color = shared feature)
7. Single HTML file - All CSS/JS inline, sortable tables, mobile-responsive
8. Do not highlight unless it is high-yield, must know, key difference, clinical pearl, or important information. The above was just an example, do not copy all of the highlights. They were too excessive.
